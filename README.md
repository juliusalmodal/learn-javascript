# Learn JavaScript

This workspace is set up to help you learn JavaScript from basics to advanced concepts.

## Learning Roadmap

1. **Basics**
   - Variables and data types
   - Operators
   - Control structures (if/else, loops)
   - Functions

2. **Intermediate**
   - Objects and arrays
   - DOM manipulation
   - Asynchronous programming (Promises, async/await)
   - ES6+ features

3. **Advanced**
   - Closures and scope
   - Prototypes and classes
   - Modules
   - Error handling

## Getting Started
Run the examples in the `basics` folder using Node.js or in a browser console.

To run a file: `node basics/variables.js`
